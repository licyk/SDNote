## 文档地址
- https://licyk.github.io/SDNote
- https://sdnote.netlify.app
- https://sdnote.vercel.app （国内访问受限）


## 将仓库克隆到本地
```
git clone https://github.com/licyk/SDNote
```


## 在本地构建文档
```powershell
# Windows CMD
.\start.bat

# Windows PowerShell
./start.ps1

# Linux / MacOS
./start.sh
```
