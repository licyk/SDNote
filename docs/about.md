# 关于
文档基于个人经验进行编写，可能存在一些不足之处。如果你有确认有误的内容或者需要补充的内容，可以发 Issue 或者提交 PR 到本仓库。

## 将仓库克隆到本地
```
git clone https://github.com/licyk/SDNote
```

## 在本地构建文档
```
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
mkdocs serve
```
