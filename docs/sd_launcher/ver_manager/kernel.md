# 内核
这是管理 SD WebUI / ComfyUI / Fooocus 内核的版本。

![kernel](../../assets/images/sd_launcher/version_manager/kernel.jpg)

这里显示了内核的远端地址、当前分支和当前版本。

刷新列表可用于刷新列表中的版本信息，一键更新可用于将内核的版本更到最新的版本。

左侧的版本 ID 点击后可查看对应版本的代码变动。

## 切换分支
!!!note
    该功能需要将绘世启动器的配置模式调整成专家模式才会显示。

切换 SD WebUI 的分支，可以将 SD WebUI 的分支在 AUTOMATIC1111/stable-diffusion-webui、lshqqytiger/stable-diffusion-webui-directml 、vladmandic/automatic 之间进行切换。

!!!note
    AUTOMATIC1111/stable-diffusion-webui、lshqqytiger/stable-diffusion-webui-directml 、vladmandic/automatic 可归类为 SD WebUI，但分支列表中还有 ComfyUI、Fooocus、油漆厂。请注意，切勿做出将 SD WebUI 切换成 ComfyUI、Fooocus、油漆厂 这些分支，或者是 ComfyUI 切换成 SD WebUI 的操作，这可能会导致环境出现损坏或者数据丢失，请对您的操作负责。
