# 简介

如果你发现网络上没有自己需要的插件或者像对一个插件修改，并且自己拥有一定的python编程基础，可以根据指南编写或修改插件。

请注意，这里只提供最基本的编写指南，所以涉及的编程语言只有python。

# 文件结构

插件可以是一个文件夹或者一个单独的文件，但是推荐用文件夹来存放你的插件文件，下面将拿我编写插件举例子。
